# AdventureWorks Analytics
 
## Overview

This repository contains an interactive Power BI dashboard for analyzing production and inventory metrics. The dataset used is based on the fictional AdventureWorks company, sourced from Kaggle. The dashboard focuses on optimizing manufacturing processes and inventory management by visualizing key metrics and trends.

## Data Source

[Data Source](https://www.kaggle.com/datasets/tituspr/adventureworks2022-excel-format)
